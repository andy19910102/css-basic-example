## 啟動方法

1. 點選頁面上方綠色按鈕「Code」，選擇「Download ZIP」取得專案壓縮檔案。
2. 將專案壓縮檔案解壓縮後使用[Visual Studio Code](https://code.visualstudio.com/)編輯器開啟。
3. 在HTML檔案編輯區按下右鍵並選擇「Open with Live Server」預覽網頁。
