顯示模式

區塊元素(block)代表: h1~h6,div,section,header,p,br,li,ul,ol...

行內文字元素(inline)代表: a, span, label

行內區塊元素(inline-block)代表: input, img, button

a

區塊元素(block)

                
選擇目標 {
    /* 設定此元素為區塊元素 */
    display: block;
}
  • 預設：
  • 會造成「」
  • 可以設定「」與「」
div
a

行內文字元素(inline)

                
選擇目標 {
    /* 設定此元素為行內文字元素 */
    display: inline;
}
  • 預設：
  • 不會造成「」
  • 不可設定「」與「」
  • 中間會有「」

a

行內區塊元素(inline-block)

                
選擇目標 {
    /* 設定此元素為行內區塊元素 */
    display: inline-block;
}
  • 預設：
  • 不會造成「」
  • 可以設定「」與「」
  • 中間會有「」
